# Мой первый веб-сервер на Python

Проект создан в Visual Studio 2026 для изучения Python.

## Структура проекта
- `main.py` - основной файл веб-сервера
- `requirements.txt` - зависимости Python
- `.gitignore` - игнорируемые файлы для Git

## Для просмотра структуры проекта используйте команды:
- tree       # показать структуру каталогов
- tree /F    # показать структуру с файлами (Windows)

## Запуск
1. Откройте проект в Visual Studio 2026
2. Нажмите F5 для запуска
3. Браузер откроется автоматически

## Функции
- Веб-сервер на порту 8000
- HTML страница с информацией о проекте
- Интерактивные кнопки
- Логирование в консоль
